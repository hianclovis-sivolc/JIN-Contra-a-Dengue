🎮 JIN contra a Dengue

Um jogo educativo desenvolvido no GDevelop com a missão de combater a dengue através da conscientização. O projeto foca no público jovem, utilizando diálogos dinâmicos e mecânicas interativas para ensinar práticas de prevenção de forma leve e dinâmica.

🚀 Destaques do Projeto
- 🌟 Apresentado no Siara Tech Summit, o maior evento de inovação, tecnológia e empreemdedorismo do Ceará.
- 💡 Mentorado pelo SEBRAE, recebendo certificação.
- 🛠️ Desenvolvido por estudantes, unindo tecnologia, educação e impacto social.

👥 Equipe

- Nayra: Responsável pela criação de toda a identidade visual, cenários e artes, aplicando seus estudos de especialização em Design.

🧠 Aprendizados e Desafios
Este projeto foi um grande marco na minha evolução como desenvolvedor e futuro gerente de projetos. Ele me permitiu aplicar na prática os conceitos de lógica de programação para resolver um problema real de saúde pública, além de gerenciar prazos e entregas para a apresentação no Summit.
